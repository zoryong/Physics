문제 PDF → Obsidian Markdown 변환기

물리 시험 PDF를 올리면 문항별로 자동 분리하여 Obsidian에 그대로 붙여 넣을 수 있는 마크다운으로 변환합니다. 텍스트 PDF는 그대로 추출하고, 스캔본은 자동으로 OCR을 수행합니다. 모든 처리는 이 브라우저 안에서만 이루어지며 파일이 외부로 전송되지 않습니다.

STEP 1
PDF 업로드
왼쪽에 시험 PDF를 끌어다 놓거나 클릭해서 선택하세요. 텍스트·스캔·혼합 PDF를 모두 지원합니다.
STEP 2
변환 실행
텍스트가 정상 추출되면 그대로 사용하고, 깨졌거나 스캔본이면 자동으로 OCR을 수행합니다. 스캔본은 시간이 더 걸립니다.
STEP 3
문항별 확인·편집
오른쪽에 문항별 마크다운이 나옵니다. 영문/수식은 자동으로 $\mathrm{}$ 형태로 변환되며, 직접 수정할 수도 있습니다.
STEP 4
다운로드
문항별 .md, 전체 physics.md, 또는 그림까지 포함한 ZIP으로 받아 Obsidian에 붙여 넣으세요.
💡 수식·그림 안내. 시험지의 수식은 전용 수식 폰트로 되어 있어 텍스트로는 읽히지 않습니다. 이 도구는 발문·보기·선지 속 인라인 수식을 원래 위치에 그림으로 잘라 ![[question03_eq1.png]]처럼 삽입하고, 문항의 그림·표·회로도는 ![[question03.png]]로 삽입합니다. 정확한 LaTeX가 필요하면 왼쪽 옵션의 AI 수식 인식을 켜서 API 키를 입력하세요(수식이 $...$로 변환됩니다).

📄 PDF 업로드

변환할 시험 PDF 파일 1개를 선택하세요.

⬆️
여기로 PDF를 끌어다 놓기
또는 클릭해서 파일 선택
A, B, p, q, n, c 등을 $\mathrm{}$로, 단위·수식을 LaTeX로 변환합니다.
문항 영역의 그림·표·수식을 이미지로 잘라 ![[questionNN.png]]로 삽입합니다.
좌·우 2단으로 편집된 시험지의 읽기 순서를 맞춥니다.
시험지의 수식은 텍스트로 읽히지 않아 기본적으로 이미지(![[..._eq1.png]])로 삽입됩니다. 체크하면 잘라낸 수식 이미지를 AI로 보내 실제 LaTeX($...$)로 변환합니다. 이 경우 API 키와 수식 이미지가 선택한 제공자에게 전송됩니다(오프라인 아님).
처리 중…
